# curso-html-aplicando-lo-aprendido-
Este repositorio contiene uno de los ejercicios que tuve en mi curso "Introducción al desarrollo WEB", en el cual tuve que aplicar todo lo aprendido en el curso introductorio (simule una tienda de venta de figuras).

Este Pull Request documenta la construcción de un catálogo visual llamado DecoFiguras, orientado a la exhibición de productos decorativos con fines promocionales. El sitio fue desarrollado con HTML y CSS puro, destacando por su estética cálida, organización clara y presentación visual efectiva.

## Componentes principales:
- `index.html`: estructura con encabezado, productos organizados en tabla e imagen destacada
- `style.css`: estilos adaptados a un catálogo comercial, con colores cálidos y tipografía accesible
- `normalize.css`: para mantener consistencia visual entre navegadores

## Características destacadas:
- Diseño responsive con imágenes centradas y adaptadas
- Paleta cromática cálida (naranjos y verdes)
- Tipografía combinada: decorativa y moderna
- Enlace externo temático que refuerza el estilo anime/pop

Este proyecto puede utilizarse como base para una tienda digital simple o un escaparate de productos.
